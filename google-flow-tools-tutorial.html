Google Flow Tools 到底是什么？

Google Flow 原本是一款面向电影创作者的 AI 视频工作室，于 2025 年 Google I/O 上正式亮相，此后已扩展到全球140+个国家。

"Image editors, video resizers, custom shaders — if you can think of it, you can now build it."
— @FlowbyGoogle 官方推文，2026年5月19日

2026年5月19日，Google 在 Flow 内部新增了 Tools 功能——你可以用自然语言描述一个 AI 工具的需求，Flow 就会自动为你创建它。图像编辑器、视频缩放器、自定义着色器……统统可以"说出来就生成"。


这不只是个小更新。这相当于 Photoshop 当年推出 Actions 的时刻——只不过现在你不需要录制宏，只需要描述你想要什么。

三大核心

本次更新的三大亮点

🛠️
Flow Tools
用自然语言创建可复用的 AI 工具预设，发布到 Tools Gallery 与社区共享，支持 Remix 再创作。无需编程经验。
🤖
Flow Agent
基于 Gemini 的创意伙伴，帮你头脑风暴、写角色对话、一次创建多个变体、批量整理资产。全部免费用户可用。
⚡
Gemini Omni Flash
全新多模态模型，支持跨场景角色一致性，可对话迭代编辑，将真实素材与生成内容无缝融合。

实际操作

从0到1：如何用 Flow Tools 创建第一个工具

以"创建一个自动把视频缩放为9:16竖屏、并保持主体居中"的工具为例：

01
打开 Google Flow，进入 Tools 面板
登录 labs.google/fx/tools/flow，找到 Tools 区域，选择「Create New」。需要有效的 Google 账号。
02
用自然语言描述你的工具需求
直接输入你想要的功能，越具体越好：
// 示例描述（推荐用英文获得最佳效果）
I want a tool that automatically crops any video to 9:16 vertical format, keeping the main subject centered in frame, with no quality loss.
03
Flow 自动生成工具，命名并保存
Flow 会生成一个有友好名称的工具，添加到你的个人库。你可以给它起一个有辨识度的名字，比如 vertical-keeper 或 9x16-auto。
04
一键发布到 Tools Gallery（可选）
如果你的工具够好，发布到 Gallery 让全球创作者使用。每次有人使用或 Remix 你的工具，你的名字都会与它相关联——这是零成本的个人品牌建设。
05
使用工具，批量处理你的内容
之后每次需要把16:9视频转为9:16，一键调用即可。结合 Flow Agent，还可以批量处理整个项目库的所有素材。
